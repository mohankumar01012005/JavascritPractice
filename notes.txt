##### DOM in javascript


      ### WINDOW VS DOCUMENT in javascript

          WINDOW:  1. window is the main container or we can say the 'global object' and any operations related to entire browser window
                      can be a part of window object.
                   2. All the members like objects, methods or properties. if they are part of window object then we dont want to 
                      refer window object.
                   3. window has method, properties and objects. Ex: setTimout() and setInterval() are the methods, where as document is the 
                      object of window and it also has a scree object with properties describing the physical display.
         
         DOCUMENT: 1. DOM is a child of window object.
                   2. in DOM we need to refer the document if we want to use the document, object, method or properties.
                   3. where as document is the object of global object that is  window, which deals with the document, the HTML 
                      elements themselves

         ==> window is a main container(parent) and  document is a child of window 


      #### DOM vs BOM

         DOM: DOM is the document model object, which deals with the document, HTML elements themselves, eg: document nad all traversal 
              you would do in it, events, etc..
            
            ==> for ex: change the background colour to red.
               document.body.style.background = "red";            // changes the background colour of body in to red

         BOM: the BOM (browser object model) is the browser object model, which deals with the browser componenets asaid from the document like history,
              location, navigator and screen(as well as some others that vary by browser ). or 
               in simple meaning all the window operations which comes under BOM are performed using BOM.

              => BOM can be used to  performs navigation, screen, location, frames, history, HTML httpREquest related  Works
      
      for example: document.getElementById()   

               -> window has methods, properties, and object.
                  ex: setTimout() or setInterval() are methods where as document is the object of window and it also has a screen 
                      object with properties discribing the physical display

               -> example of window object properties are innerHeight, innerWidth and many more

     ## FUNCTIONS alert/confirm/prompt are also part of BOM:
